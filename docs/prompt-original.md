# 命题原文

> 本文件存放比赛命题的原文摘录，避免反复向他人索要。仅摘录与方案设计直接相关的"三个真实挑战"与"AI 机会"段的四个要求，非逐字全文。

---

## 三个真实挑战

1. **洞察深度与覆盖面不足**：最终分析依赖研究员主观判断，难以覆盖海量 UGC 内容，细分洞察颗粒度粗，且知识无法沉淀复用——每次研究成果散落在 PPT/文档中，更多体现在公司管理层流转，而不能向员工覆盖。
2. **数据结论准确率低**：基于分析的用户画像形成的产品决策，在产品上市后会发现有较多偏差，通常体现在实际产品用户与画像用户群并不高度统一、画像推荐的卖点与实际市场反馈不一致等，如何建立一套可被验证的画像机制，提升产品决策的准确性。
3. **竞品追踪能力薄弱**：受人力限制，过去只能研究少数核心竞品，无法对 10+ 竞品车型的用户画像进行持续动态追踪与对比。

## 方案需涵盖的四个要求（出自"AI 机会"段）

1. 多源数据的智能采集与融合
2. 多维度人群地图的动态构建（支持按人生阶段 × 城市线级 × 年龄 × 性别等维度实时下钻）
3. 产品经理可随时调用的 AI 问答洞察能力
4. 基于真实用户画像的车型模拟数字人原型

## 目标句

将用户研究的响应周期从"等报告"（数周）压缩至"即时查"（分钟级），让每一款新车从立项起就精准命中目标人群。
